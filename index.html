O que é essa máquina?

Pense nela como um cérebro eletrônico que recebe, armazena e processa informações para nos ajudar.

É composta por duas partes essenciais:

  • Hardware (A parte física): É tudo o que você pode tocar! Monitor, Teclado, Mouse e a CPU.
  • Software (A parte "inteligente"): São os programas e aplicativos que dão vida ao hardware. Windows, Word, Google Chrome.

Uma Viagem no Tempo

Os primeiros computadores, como o ENIAC (1945), eram gigantescos, ocupavam salas inteiras, pesavam 30 toneladas e serviam para cálculos militares complexos.

A programação nasceu da necessidade de "dar ordens" a essas máquinas. Uma das pioneiras foi Grace Hopper, que criou as bases para linguagens mais próximas do inglês.

Curiosidade: O termo "bug" (inseto) para se referir a uma falha de sistema foi popularizado por Grace Hopper, quando ela encontrou uma mariposa de verdade presa nos relés do seu computador, causando um mau funcionamento!

Programar vs. Desenvolver

O que é Programação?

É a arte de escrever instruções (código) que um computador entende e executa. É como ensinar um idioma novo para a máquina, usando linguagens como Python, JavaScript ou C#.

Analogia: Programar é fabricar os tijolos.

O que é Desenvolvimento de Sistemas?

É o processo completo de criar uma solução. Envolve entender um problema, planejar, programar (usando os tijolos), testar e entregar o software pronto (a casa construída).

Analogia: Desenvolver um sistema é projetar e construir a casa inteira.

Desvendando Mitos: O que é e o que NÃO é Desenvolvimento de Sistemas

✔️ O que É

  • Resolver problemas reais com tecnologia.
  • Trabalhar em equipe e colaborar.
  • Aprender constantemente.
  • Criar ferramentas que facilitam a vida.
  • Ser criativo e lógico ao mesmo tempo.

❌ O que NÃO É

  • Ser um "hacker" de filmes.
  • Trabalhar sozinho em um porão escuro.
  • Apenas "formatar o PC da tia".
  • Saber consertar qualquer eletrônico.
  • Apenas digitar códigos sem parar.

As Ferramentas do Ofício

Para programar, não usamos o Word. Usamos Editores de Código ou IDEs (Ambiente de Desenvolvimento Integrado).

Um editor moderno como o VS Code é como uma oficina cheia de superpoderes:

  • Syntax Highlighting: Colore o código para facilitar a leitura.
  • Autocomplete: Sugere comandos para você não ter que decorar tudo.
  • Depuração (Debugging): Ajuda a encontrar e esmagar os "bugs"!
  • Extensões: Permitem adicionar novas ferramentas e customizar seu ambiente.

Sua Jornada de Aprendizagem

Estas são algumas das matérias essenciais que vocês aprenderão:

  • Lógica de Programação: É o "como pensar" antes de escrever o código. Importância: É a base de tudo, como a gramática para um escritor.
  • Banco de Dados: Onde e como as informações (de usuários, produtos) são guardadas. Importância: Sem ele, seu app teria amnésia!
  • Front-End: A parte do sistema que o usuário vê e interage (telas, botões, menus). Importância: É a "vitrine" do seu projeto.
  • Back-End: O "cérebro" do sistema que funciona nos bastidores. Importância: É o motor que faz tudo acontecer.
